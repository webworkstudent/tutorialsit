    • Array
    Array is a collection of similar type of elements which have contiguous memory allocation.

    1. Array is based on indices [ indexes ].
    2. Array Can hold only homogeneous element. [homogeneous = similar type.]
    3. Array have a fixed Size.
    4. Array always start from 0 index.
    5. Array always Ends with n-1 index;
    6. n is a size of an array.
    Example:1
    Type of an Array
     1. One Dimensional array Or linear Array OR vector
    2. MultiDimensional array Or matrix OR Two Dimentional

    Declaration of one Dimensional array
     DataType variableName[size];
    int marks[5];
    float marks[6];
    char name [30];

    Declaration of Two Dimensional array
     DataType variableName[row][column];
    int marks[3][3];
    float marks[2][3];
    char name [3][2];

    [0] = Starting Index
    [4] = end Index
    12 = value at index [1]
    5 = size of an array.
