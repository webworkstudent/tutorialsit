    Type of variable
     1. global variable -- The variable which is declared outside any function.
    2. local variable -- The variable which is declared in side a specific function.
    3. static variable -- The variable which is declared as static know as static variable. it get's memory only once.
    4. constant variable -- whose value never change.

    Control structure -- is a technique in which a computer takes decision itself for next step.

    Way for control the program
    1. if --- is a keyword which is execute
    when the condition is true.

    2. if else - is a technique in which if condition is
    true than if will execute otherwise else will execute.

    3. else if - is a technique in which if condition become
    false then else if will execute to check for next condition.

    4. nested if -- one if inside another if know as nested if.
