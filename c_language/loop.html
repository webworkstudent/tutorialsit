    • Loops
    loops are repetitive structure. It will iterate the code which exist into the block until the condition remains true.

    Type of the loops
    1. condition Control -- Loop which is controlled by a condition is known As Condition Control.
    Like : while, do while .
    2. Counter Control -- The Loop which is controlled by a counter is know as counter control loop.
    Like : For loop
    Condition Control
    1. while -- condition will be checked first. also called Entry Control loop.
    --Syntax for while--
    ============================
    while(condition) // Entry control loop
    {
    // statements;
    }
    do while -- condition will be checked later on. so it is called exit control loop.
    at least once this loop will execute even condition is false.
    --Syntax for do while--
    ============================
    do
    {
    // statements;
    }while(condition); // Exit control loop
    Counter Control Loop
    1. for -- it is based on counter.

    Syntex for for loop
    ============================
    for(initialization;condition;increment/)
    {
    // statements;
    }

    For LOOP -- it is counter control loop. in for loop we must have initial value and initial value will increment by one or any number until it not exceed the condition number.

    Syntax of the for Loop
    for(initialization; condition ; increment/decrement)
    {

    // statements;
    }
    for : - is a keyword.
    () - is a signature or parameter block.
    {} - body or scope of the for loop.
