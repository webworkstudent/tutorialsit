    • Pointers

    - Pionter is a special type of variable which is used to hold the address of any other variable.


    Declaration of Pointer variable


    Datatype * variableName ;
    DataType -- is a type of data. like int , float , char, double etc.
    * / astrik / -- represent the variable pointer means location of particular
       variable OR Address OR pointer. variableName --any name for variable.like firstName, lastName , roll ,e_id etc
    int x =5; // Normal variable declaration
    int * y;// pointer variable declaration

    Assignment of pointer variable

    y=&x;

    How the memory get allocated to the pointer variable

    x    name of the variable
    | 5 | value of the variable
    -----
    1234 Address of the variable
    y name of the variable
    ------- |1234 | value of the variable
    ------- 1238    Address of the variable

    Like nested value do we have any nested address Assignment

    x=y=z=5;    // PRINTF("%D",X); //5
    int *PTR1,*PTR2,*PTR3;
    PTR1=PTR2=PTR3=&X; //PRINTF("%D",PTR1); //5 // 1234
    Nested Pointer to access same value from all:
    int *PTR1,**PTR2,***PTR3; // pointer variable declaration
    X     PTR1     PTR2     PTR3
    ------- ---------- -------------- --------------
    | 5 |    |&X (1234)|     |&PTR1 (4321)|    |&PTR2 (5432)|
    ------- ----------- -------------- --------------
    1234 4321 5432 9876
