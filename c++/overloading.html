    • Operator Overloading
    Overloading
    Refers to the use of the same things for different purposes. Operator overloading : An operator can be redefined to perform additional tasks.
    Example :
    Consider the operation of operator(+). Operation is sum if operands are numbers and if type is string then operation is string concatenation. However using operator overloading you can extends the capability of + operator so that this operator performs the addition of two objects of a class.
    Syntex of Operator Overloading function
    Return_type operator op (op-arguments)
    {
    // function body
    }

    return_type : is the data type returned by the operation.
    operator : is a keyword.
    op : is the overloading operator like +, - etc
    op-arguments : is the list of arguments.

    Operator Overloading For Member function

    A member function does not provide any argument for unary operation and it has only one argument for binary operation whereas a friend function has one argument for unary operation and two arguments for binary operation.
    Example :
    void operator -(); // Unary minus with no return value.
    Complex operator + (complex); // binary + used to add two complex numbers
    friend complex operator + (complex, complex); // binary + using friend function.
    Overloading Unary Operators
    Unary operators act on only one operand.
    Like :
    - (unary minus)
    ++ increment
    -- decrement
    etc
    Example:1
    Overloading Unary minus 
    #include<iostream>
using namespace std;

class sample
{
  int m;
  int n;
  public :
  void getData(int a,int b);// Declaration
  void display();
  void operator -(); // overload unary minus
};
// :: scope resolution operator
void
sample :: getData(int a,int b) // Definition
{
m=a;
n=b;
}

void sample :: display(void)
{
 cout<<m<<" "<<n<<endl;
}

void sample :: operator -()
{
  m=-m;
  n=-n;
}

int main()
{
  sample obj; // object creation of sample class
  obj.getData(40,-25);
  cout<<"\n The value of the data member Before operator overloading \n";
  obj.display();
  -obj; // Operator overloading
  cout<<"\n The value of the data member after operator overloading \n";
  obj.display();
   return 0;
}

    Overloading the I/O operators
    #include<iostream>
#include<string.h>
 using namespace std;
 class sample
 {
   int num;
   public:
     sample()
     {
       num=0;
     }
     friend ostream& operator << (ostream&, sample&);
     friend istream& operator >> (istream&, sample&);
 };



 // Overload insertion operator function
 ostream& operator << (ostream& out,sample& no)
 {
   out<<no.num;
   return out;
 }

  // Overload extraction operator function
 istream& operator >> (istream& in,sample& n)
 {
   in>>n.num;
   return in;
 }
