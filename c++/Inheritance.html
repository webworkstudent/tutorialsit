    • Inheritance?
    Inheritance is a mechanism of inheriting the property from parent class object to child class object.

    what is property?
    • Variables and functions are know as properties.
    • ==> Parent Class ==> super class ==> Base class
    • ==> Child class ==> Sub class ==> Derived class

    Type of Inheritance.


    There are five type of inheritance :-
    • Single Inheritance : when class B inherit the property of class A is know as Single Inheritance.
    • Multilevel Inheritance : When class C inherit the property of class B and class B inherit the property of class A these levels are know as multilevel Inheritance.
    • Multiple Inheritance : when we have more than one parent of a class is know as multiple Inheritance.
    • Hierarchical Inheritance: when a class have more than one child is called Hierarchical inheritance.
    • Hybrid Inheritance : The Combination of more than one inheritance know as Hybrid inheritance.
